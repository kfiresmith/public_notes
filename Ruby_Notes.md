# Ruby Notes

### Basics
#### Printing (puts)
```ruby
puts "Some string to putput"
```
#### User Input (gets)
```ruby
input = gets
```
#### Variable Types and Formatting
**Arrays**
```ruby
my_empty_array = []
my_array = [0,1,2,3,4]

```
**Hashes**
```ruby
my_hash = {"key" => "value", "old bannana" => "tasty"}

puts my_hash["old bannana"]
>tasty
```
#### Logic
**If, elsif, else, end**
```ruby
if my_var > 10
  ...
elsif my var < 5
  ...
else
  ...
end
```
